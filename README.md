# pymodoro.py

A gui pomodoro timer written in python.

**Dependencies:**
* tkinter (for gui)
* ffplay (from ffmpeg for playing notification sound)
* notify-send (for popups)

**Keyboard shortcuts:**
* subtract one minute from countdown (<Right>)
* add one minuter to timer (<Left>)
* Start/stop (<space>)
* Reset (r)
